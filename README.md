# Strona Karoliny

Jest to storona wizytówka.

Funkcjonalności:

- foto w tle statyczny content
- ładny pasek menu
- blog/wipsy

Cechy:

- będzie od zera
- na google firebase będzie bazować

## Setup

```shell

npm run storybook # storybook
npm start         # apka
npm t             # testy

```
